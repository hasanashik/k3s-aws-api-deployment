## Python REST API with Flask
api/\
├── app.py\
├── Dockerfile\
└── docker-compose.yml
```bash 
apt install systemctl
pip install Flask 
apt install docker.io
sudo systemctl start docker
```
Build the Docker image
docker build -t flask-api .
```bash 
docker-compose up
```
